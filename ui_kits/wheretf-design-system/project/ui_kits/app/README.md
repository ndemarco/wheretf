# WhereTF · App kit

React recreation of the product app. Four navigable screens, click-through only.

- `AppSidebar.jsx` — fixed sidebar with workspace + nav + user
- `FindScreen.jsx` — the main verb: search items/locations/codes
- `ItemsScreen.jsx` — deep catalog with category filter
- `ModulesScreen.jsx` — storage-unit browser + bin grid
- `ScanScreen.jsx` — barcode scan with recognition → link flow

No real data. `SEED_ITEMS` lives on `window` in `FindScreen.jsx`.
